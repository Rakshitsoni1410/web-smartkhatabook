# Smart Khata Book

Smart Khata Book is a modern business management system for **Retailers** and **Wholesalers**.
It includes a **Web Panel**, **Backend API**, and **Mobile App Repository** for future expansion.

---

# 🚀 Repositories

## 🌐 Web Frontend Repository

React.js admin/business dashboard.

```text
smart-khata-web
```

## ⚙️ Backend Repository

Node.js + Express.js + MongoDB API.

```text
smart-khata-backend
```

## 📱 Mobile App Repository

Flutter app for Android / iOS users.

```text
smart-khata-app
```

---

# ✨ Features

## 🔐 Authentication

* User Registration
* Secure Login
* Role Based Access

  * Retailer
  * Wholesaler

## 📦 Stock Management

* Add Product
* Edit Product
* Delete Product
* Search Product
* Category Management

## ⚠️ Inventory Alerts

* Low stock warning at quantity **5 or less**
* Reorder suggestions

## 🤝 Supplier Marketplace

Retailers can:

* View wholesalers by category
* Order extra stock
* Find nearby suppliers

## 📊 Dashboard

* Product stats
* Orders summary
* Business cards
* Clean analytics UI

## 📱 Mobile App (Planned / Repository Ready)

* View stock
* Receive notifications
* Place orders
* Quick dashboard

# ⚙️ Installation

## Web Frontend

```bash
cd smart-khata-web
npm install
npm run dev
```

## Backend

```bash
cd smart-khata-backend
npm install
npm run server
```

## Flutter App

```bash
cd smart-khata-app
flutter pub get
flutter run
```
